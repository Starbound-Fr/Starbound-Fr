Il est si bon, le savane...

September, 03 2013 Oniricorpe

Salut les gars ! Ça faisait longtemps que je n'avais rédigé un article, mais deux seront postés dans la journée ! En attendant le second, je vous laisse apprécier le premier, rédigé par Tiy. Ce n'est pas dans les habitudes de l'équipe de nous fournir une liste de ce genre, mais bon nous ne faisons que relayer !
- Nous avons terminé de travailler sur le biome "savane", incluant de nouveaux arbres et feuillages. - Quelques améliorations sur le boss OVNI. - Nous avons retouché le code lié aux mouvements, pour l'implémentation du système de tech. - Des améliorations ont été apportées à divers endroits de l'interface graphique. - Les mobs volants ont reçu de nouvelles animations pour leurs attaques. - De nouveaux vêtements ont été ajoutés. - Le jeu mémorise désormais si sa fenêtre avait été maximisée juste avant de la quitter. - De nouveaux paternes d'IA ont étés ajoutés. - Nous travaillons actuellement sur les donjons. - Pas mal de correction de bugs et d'améliorations de performances réalisés. (ndlr : Sony-like :troll: )
Personnellement, j'aime beaucoup moins ce genre d'actu "listés", mais c'est sûrement signe d'un manque de temps. Néanmoins, ils savent se rattraper avec une simple image :) Edit : Tiens, un SMS d'Emocaplin ! Il semblerait que je l'aie pris de court pour la rédac' de cet article ! ^w^ Source : [2nd September Progress]
